# Employee Management System

## Description
This is a console-based Employee Management System built using Core Java. It demonstrates OOP principles, CRUD operations, and file handling with Java Serialization.

## Features
- **Employee Management:** Add, update, delete, and list employees.
- **Department Management:** Manage department roles and assignments.
- **Data Persistence:** Saves data to files so records are not lost after restart.
- **Reporting:** Calculate salaries and generate simple reports.

## Tech Stack
- Java (JDK 17+)
- Maven (Build Tool)
- IntelliJ IDEA

## How to Run
1. Clone the repository.
2. Open in IntelliJ IDEA.
3. Run the `Main.java` file.
